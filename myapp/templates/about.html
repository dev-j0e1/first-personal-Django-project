About Django App

Project Overview

Django App is a demonstration project showcasing a secure, modern web application built with Django. It implements industry-standard authentication patterns with proper security practices. In all honesty. This project was largely vibe-coded. I tried to learn as much as I could. But I procrastinated a little too much. So I had to rush this project. I struggle with little direction. That is something I will need to work on quite a lot. But I am prepared to do much better. This was just a bit of a wake up call for me that I'm not as compitent as I think I am just yet and that it will require a lot of work and time to get to that point.

Technologies Used
  • Backend: Django 6.0.3
  • Frontend: Bootstrap 4.3.1
  • Database: SQLite
  • Authentication: Django's built-in auth system with PBKDF2 hashing
  • Language: Python 3.14
Security Features
  • Email-based authentication with hashed passwords
  • CSRF protection on all forms
  • Session-based user authentication
  • Secure password validation using Django's default PBKDF2
  • HTTP-only session cookies
Key Features
  • User login and logout functionality
  • Responsive design for all screen sizes
  • Message display system for user feedback
  • Navbar with dynamic user state display
  • Clean, intuitive user interface
Getting Started

To get started with this application, navigate to the Login page to authenticate with your email and password. Once logged in, you'll have access to your user profile and can manage your session.
